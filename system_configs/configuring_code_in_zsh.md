<h2>[MAC-specific] Configuring code in zsh to open VScode from cd using code .</h2>
-------------------------------------------------------

<h3>Step by Step Guide</h3>
-------------------------------------------------------
<h4>Step 1: Confirm Visual Studio code is installed in your system</h4>
- Go to the official Visual Studio Code website or search for "VS Code download" on Google.
- Click the download button for your operating system (Windows, macOS, or Linux).
- Locate the downloaded file in your downloads folder and double-click it to start the installation wizard. 

<h4>Step 2: Install code command in PATH</h4>
- Open Visual Studio Code.
- Bring up the Command Palette by pressing 'Cmd + Shift + P'.
- Type and select Shell Command: Install 'code' command in PATH.

<h4>Step 3: Restart Terminal and launch VS code</h4>
- Close the terminal and reopen it.
- Try navigating to the desired file dir using cd.
- Type 'code .' to check if VS Code opens.
